### Free Stuff



#### Dev Environment Setup
1. Clone to local workspace

2. Frontend
  - `cd frontend`
  - `npm install`
  - add `firebase.js` to `frontend/src`

3. Backend
  - `cd backend`
  - `npm install`
  - add `.env` to `backend`
    - Set `DATABASE_URL="<you_mongo_connection_string>"`
   


- Install 'Prettier - Code formatter' extension in VSCode
